# Menu Pick 📍

A full-stack `Java web application` that allows customers and restaurant owners to
search for restaurants, make or manage reservations and leave reviews.

---

## Tech Stack

### Back-end

- Java
- Lombok
- JSP & Servlets

### Front-end

- HTML
- CSS
- JavaScript
- jQuery
- AJAX

### Database

- Oracle Database

### Server

- Apache Tomcat

### External APIs

- Naver Email
- Kakao Map
